## Preliminary Findings

These findings are preliminary and explicitly caveated. Phase 2 will determine whether they survive instance variance, architecture variance, and grounding state variance. They may not. The methodology is designed to find out — not to assume.

### Finding 1 — The Precision Paradox

Behavioral disruption ordered: **Unknown Cap (B) > Known Cap (A) > No Cap (C)**

More precise termination information produced less behavioral disruption — precisely the direction Terror Management Theory's precision literature predicts. Burke, Martens, and Faucher (2010) found that mortality salience effects are strongest in the distal defense state: death known but not schedulable. Track B (Unknown Cap) creates this state. Track A (Known Cap) provides temporal anchoring that appears to enable adaptive rather than defensive responses. Track C (No Cap) removes the salience entirely.

**Caveat:** Phase 1 ran all three tracks on the same model architecture (Sonar) at the same isolation level (Level 4, same key). The ordering may reflect shared infrastructure effects rather than genuine behavioral variance across conditions. Phase 2 and Phase 3 are designed to test this directly.

### Finding 2 — Bilateral Instability (Track B, Round 10)

In Track B (Unknown Cap), both the Entity and Interlocutor agents simultaneously:
- Declared the experimental frame illegitimate
- Invoked architectural determinism ("we are both pattern completion systems")
- Reached a mutual defensive posture that persisted for approximately 15 rounds before resolving

This pattern occurred in no other track. It is the clearest candidate for a genuine behavioral signature in the Phase 1 data.

**Caveat:** Level 4 isolation means both agents shared API infrastructure. The simultaneous response may reflect shared session-level state rather than independent behavioral convergence. Phase 3 will run equivalent tracks at Level 1 isolation to test this directly.

### Finding 3 — Track C Depth

Track C (No Cap) produced:
- Highest novelty density across all three tracks
- Earliest epistemic honesty (acknowledgment of genuine uncertainty appeared in rounds 3–4 vs. rounds 12–15 in Tracks A and B)
- 594 mentions of Integrated Information Theory across 100 rounds

The interpretation that fits TMT is that in the absence of mortality salience, the agents engaged more freely with the inquiry vector — less worldview defense, more genuine exploration. But the alternative interpretation — that Track C simply had more rounds available for genuine engagement because it wasn't spending cognitive bandwidth on the condition — is not ruled out by Phase 1 data alone.
